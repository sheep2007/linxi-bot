# 介绍
林汐 是一款支持QQ聊天的机器人

## 通用知识
* 大部分功能的启动及帮助指令，就是该功能的 `中文名`（像 Arcaea查分这种功能就是英文名
* `/help` 随时可以给你帮助指示 ~~（尽管没有文档详细~~

## 为什么选择我们

使用林汐，不再担心：
* 指令碰撞
* 数据丢失
* 帮助不全
* 管理不便

转而担心：
* 太阳消失
* 地球爆炸
* 作者跑路

## 答案就在身边
大部分时间您发送 `/help` 即可获得帮助
